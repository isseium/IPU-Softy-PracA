\section{EPS形式への変換、および印刷}
\begin{itemize}
\item 変換手順
\begin{enumerate}
\item bmpファイルからpbmファイルに変換する。
\item pbmファイルからpgmファイルに変換する。
\item pgmファイルからepsファイルに変換する。
\end{enumerate}
\item 印刷手順
\begin{enumerate}
\item 対象のepsファイルをghostviewで開いて内容を確認する。
\item lprコマンドで対象のepsファイルを印刷する。
\end{enumerate}
\end{itemize}

すなわち次のコマンドを実行する。
\begin{screen}
\$ bmptoppm picture.bmp \textbar \ pnmtopgm \textbar \ pnmtops \textgreater \ picture.eps \\
\$ ghostview picture.eps \& \\
\$ lpr picture.eps 
\end{screen}

\section{xv,gvコマンドについて}
\begin{itemize}
\item xv\\
様々な画像処理を行うことができるソフトウェア。\\
ファイル形式の変換や、画像の拡大縮小、切り抜きなどの操作が可能である。\\
またスクリーンダンプも可能である。\\

\item gimp\\
多機能なフリーの画像処理ソフトウェアである。ブラシなどの高機能なペイントツールが多数搭載されている。また同時にたくさんのファイルを開き編集することも可能である。\\
さらにプラグインを追加することで画像処理を行う機能を追加することも可能である。\\

\end{itemize}

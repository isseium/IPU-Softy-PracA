\section{\LaTeX 文書の作成１}

指示された文章を以下に示す。

\begin{screen}
\ \ LaTeX を用いて文章を作成する手順は以下の通り。\\
1. muleなどのエディタで文章を入力する。\\
2. platexコマンドを利用してDVIファイルを作成する。\\
3. 整形された文章をxdviコマンドでプレビューしてみる。\\
4. dvipsまたはdvi2psコマンドを利用してPSファイルを作成する。\\
5. lprコマンドで印刷する。
\end{screen}
